Yêu cầu để khởi chạy bài thực hành:
Nodejs

Cài đặt các thư viện dùng trong bài thực hành:
npm install --save three dat.gui vite

Để chạy bài thực hành:
npx vite

Thay đổi các thông số của camera
Thay đổi thông số LookAt thay đổi ở phần Camera properties/LookAt
Thay đổi thông số VRP thay đổi ở phần Camera properties/VRP 

Thay đổi các thông số của spotlight ở light properties

Thay đổi các thông số của plane ở plane properties

Thay đổi các thông số của sphere ở sphere properties
